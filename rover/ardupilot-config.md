# Ardupilot configuration

## Arming

**ARMING_CHECK**: 9376  
Only check Parameters, Board voltage, Loggig, and system.

**BRD_SAFETYENABLE**: Disabled  
Disable Pixhawk physical safety switch (not connected).

## I/O

**RELAY_PIN**: AUXOUT5  
**RELAY_PIN2**: AUXOUT6  
Track direction output. See [Rover > Pixhawk]()
